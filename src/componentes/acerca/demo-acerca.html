🚀 La Visión y Filosofía

MitaDOM no es solo código, es una declaración de intenciones. Creado con la firme convicción de que la web moderna no necesita ser pesada ni depender de cajas negras para ser interactiva, veloz y escalable.

⚡ ¿Por qué "MitaDOM"?

La combinación: Une Mitay (en quechua, el sistema de turnos de trabajo optimizado en el Potosí colonial) + Mita (en guaraní, que significa "niño/nuevo/nacimiento").

Lo que comunica: Significa "Ciclo de renovación eficiente".

Justificación técnica: Expresa con precisión el ciclo de vida de tus Custom Elements y Signals. Modifica el DOM por "turnos" exactos y optimizados (quechua) para hacer nacer elementos web nativos y limpios (guaraní).

🏆 ¿Por qué elegirnos frente a la competencia?

A diferencia de frameworks pesados basados en Virtual DOM (React, Vue), MitaDOM trabaja **directamente con las APIs nativas del navegador**.

  • Zero Dependencies: Carga ultrarrápida sin node_modules masivos.
  • Reactividad Quirúrgica: Los Signals actualizan solo el textNode o atributo necesario (0% re-renders de Layout).
  • Curva de Aprendizaje Plana: Si sabes HTML, JS y Web Components, sabes MitaDOM.
